## Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Run build the project:

```bash
npm run build
# or
yarn build
```

Start the built the project:

```bash
npm run start
# or
yarn start
```

Run the jest tests:

```bash
npm run test
# or
yarn test
```

Run the type check:

```bash
npm run type-check
# or
yarn type-check
```

Run jest and type checks:

```bash
npm run test-all
# or
yarn test-all
```
